# Hailo Vision — Roadmap

## Phase 1: Foundation (Q2 2026)
- [RC] GStreamer + Hailo-8 inference pipeline
- [RC] Camera manager for RTSP, USB, CSI streams
- [RC] YOLOv8 object detection at 30 FPS
- [RC] Basic detection event dispatch (webhook, MQTT)
- [RC] CLI tool for camera config and pipeline management
- [RC] Landing page live at hailo.blackroad.io

## Phase 2: Intelligence (Q3 2026)
- [RC] Face detection (RetinaFace) and recognition (ArcFace)
- [RC] Object tracking (DeepSORT) across frames
- [RC] Detection zones with polygon boundaries
- [RC] Time-based alert rules and schedules
- [RC] People counting with entry/exit logic
- [RC] SQLite event log with full-text search

## Phase 3: Analytics (Q4 2026)
- [RC] Heatmap generation from detection data
- [RC] Path tracking and trajectory visualization
- [RC] Grafana dashboard templates
- [RC] Dwell time analysis at zones
- [RC] Traffic flow modeling
- [RC] Daily/weekly automated reports

## Phase 4: Fleet Scale (Q1 2027)
- [RC] Multi-node fleet management via WireGuard
- [RC] Centralized camera dashboard
- [RC] Model distribution and remote updates
- [RC] Aggregated cross-site analytics
- [RC] PostgreSQL backend for large deployments
- [RC] Kubernetes deployment for enterprise

## Phase 5: Custom Models (Q2 2027)
- [RC] Custom model training pipeline (PyTorch to HEF)
- [RC] Transfer learning toolkit for domain-specific detection
- [RC] Model marketplace for community models
- [RC] AutoML for edge model optimization
- [RC] Pose estimation and action recognition

## Long-Term Vision
- [RC] Every BlackRoad device can see and understand its environment
- [RC] Privacy-first computer vision as the standard, not the exception
- [RC] Bridge physical and digital worlds through sovereign edge AI
